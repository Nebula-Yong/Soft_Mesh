# soft_mesh

#### Description
使用软件定义Mesh网络，给底层硬件不支持IEEE 802.11s标准但是硬件上可以实现WiFi STA和AP共存的开发板提供软件Mesh方案。抽象了硬件接口层，方便移植和应用到其他通信方式上，如蓝牙、星闪等。
